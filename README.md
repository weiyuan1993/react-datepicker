# React DatePicker

A simple DatePicker built by pure React + TypeScript without CRA and other DatePicker packages

## Get Started

```bash
cd /path/to/project
npm install
```

## Development

```bash
npm start
```

## Demo

```bash
npm run demo
```

## TS Check

```bash
npm run ts-check
```

## Build

```bash
npm run build
```
